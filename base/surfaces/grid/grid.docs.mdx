import {Grid} from './grid';

## React Component for rendering text

A basic div that renders some text

### Component usage
```js
<Grid container>
    <Grid item>
        <div>Item 1</div>
    </Grid>
</Grid>
```

### Using props to customize the text

Modify the text to see it change live:
```tsx live
() => {
    return(
      <Grid container spacing='lg'>
        <Grid>
            <div>Item 1</div>
            <div>Item 2</div>
        </Grid>
    </Grid>
    );
}
```
